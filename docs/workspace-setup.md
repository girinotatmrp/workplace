# Team Workspace

Simple documentation editor + architecture diagrams with GitHub backup.

## Features
- Markdown editor
- Excalidraw diagrams
- Auto-save to MongoDB + GitHub

## Auto Deploy

Just push to GitHub - backend starts automatically at https://api.notatmrp.com/api/workspace

## How to Use

1. Go to https://main.d1rtebrcdf2t12.amplifyapp.com
2. Click "Team Workspace" button
3. Create docs or diagrams
4. Click save → syncs to GitHub
 
Done! 🚀
